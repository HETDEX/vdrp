Welcome to VDRP
========================================================================

Version: |release|

``VDRP`` the Virus Data Reduction Pipeline is a collection of scripts
and FORTRAN programs for astrometry, throughput and flux
limit calculation.

``VDRP`` currently supports only python 2.7.

.. This documentation is also available on `fplaneserver.readthedocs.io
.. <https://fplaneserver.readthedocs.io>`_.
